How to Change a Folder's Icon using Terminal

By: Ila Wallace - Difficulty: Medium - Time Required: 45 minutes

Introduction

This tutorial will teach you how to change the default folder icon into your own custom folder icon using Terminal in Mac OSX.

After the completion of this tutorial, you will be able to navigate through the Mac OSX directory using Terminal and assign your own custom .png image as a folder icon.

Requirements

To complete this tutorial, you will need access to a Mac OSX computer with at least 5 GB of available memory space.

Additionally, you will need to Xcode Command Line Tools installed on your computer. If you do not already have it installed, the section Installing Xcode Command Line will guide you through the installation process.

Lastly, you will need to create a new folder and put a custom icon .png file of your choice inside the folder. If you do not have an image file yet, you may choose from one of the default options provided in the section Downloading a Custom Icon or search the internet for one.

Warning: If you have any important files on your desktop, make sure to back up these files in a separate location like on a flash drive or in OneDrive before proceeding any further. Using Terminal can change your computer's file system and its data.

Introduction to Terminal

This section covers important concepts regarding using Terminal as well as providing a reference table of Terminal commands that will be used in this tutorial.

Table of Terminal Commands

This reference table lists all terminal commands that appear in this tutorial and states briefly what they do.

Terminal Commands Description
xcode-select --install Installs the Xcode Command Line Tools
xcode-select -v Prints the current version of installed Xcode Command Line Tools
pwd Prints the current location's address
cd ~; pwd Moves to the user's home directory and prints the location's address
cd /Desktop/custom Moves to the custom directory
sips -i icon.png Formats image .png file to icon .png file
DeRez -only icns icon.png > tmpicns.rsrc Converts .png file to .icns file referenced through the tmpicns.rsrc file
Rez -a tmpicns.rsrc -o Icon$'\r' Generates Icon? file from .icns reference file
SetFile -a C . Sets Icon? file as the folder's custom icon
SetFile -a V Icon$'\r' Hides the Icon? file inside the folder
rm tmpicns.rsrc icon.png Removes unecessary leftover files
rm Icon$'\r' Deletes the custom icon from the folder

Getting Started

This section covers the preliminary tasks required for this tutorial, which are:

  1. Creating a new folder on your desktop and renaming it custom

  2. Downloading a custom icon image file and saving it in the custom folder as icon.png

  3. Opening a new Terminal window

  4. Installing Xcode Command Line Tools onto your computer


Note: For the purposes of this tutorial, the folder will be refered to as custom and the icon .png file as icon.png. These files can be named anything; however, it is recommended to only change the custom folder name once you have completed the tutorial as Terminal syntax errors can occur when using special symbols.


When all preliminary tasks have been completed, you can move onto the section on how to navigate through Mac OSX's directory file system.


Creating a New Folder

To create a new folder on your desktop, you must do the following:

  1. Navigate to your desktop home screen

  2. Move your cursor to an empty location and then press two fingers down on the track pad, or if you have an external mouse, right click on the folder to select it

  3. Click on the option called New Folder


A blue default folder named untitled folder will now appear on your desktop screen.


To rename the new folder, you must do the following:

  1. Move your cursor over the folder and then select the new folder

  2. Click on the option called Rename

  3. Type the following text into highlighted folder name text box and then hit Return↩︎:

    custom

Checkpoint: A new folder named custom should now be on your desktop screen.


Downloading a Custom Icon

In order to change the default folder icon, we need to have a new icon file to change it to.

If you have a custom icon downloaded to your computer, move a copy of the image to the icon to your desktop and name it icon.png. Then, you can skip to Installing Xcode Command Line Tools.

If you do not have a custom .png file on hand, you can download an icon from here. Make sure to name it icon.png and save it in your desktop folder custom.


Checkpoint: The custom image should now be inside the custom folder.


Opening a New Terminal Window

To open the Terminal application, you must do the following:

  1. Press both the Command ⌘ and Space bar keys on your keyboard at the same time to open Spotlight Search

  2. Type the following text into the Spotlight Search prompt:

    Terminal

  3. Press the Return↩︎ key on your keyboard.


Checkpoint: A new Terminal window should be open and visible on your screen.


Installing Xcode Command Line Tools

To install Xcode Command Line Tools onto your computer, you must do the following:

  1. Open a new Terminal window

  2. Type the following text into the prompt:

    xcode-select --install

  3. When the pop up appears asking you if you would like to install the tools, click on the option Install.

  4. When the pop up appears stating the software was installed, click on the Done option.


If you would like to confirm your download, you can do the following:

  1. Open a new Terminal window

  2. Type the following text into the prompt and then press Return↩︎:

    xcode-select -v

The resulting output prints the current version of your Xcode Command Line Tools where # represents a version number:

xcode-select version #.

Checkpoint: Xcode Command Line Tools should be now be installed on your computer.

Navigating a Directory

When using Terminal for any task, it is important to have a basic understanding of where our current location in the Mac OSX directory file system and how to move through the directory file system.

If you know how to navigate a directory file system in Terminal, you may skip to Changing a Folder's Default Icon.


Identifying the Current Location

To identify our current location in the directory tree, we can use pwd.

Type the following text into the terminal prompt and then hit Return↩︎:

pwd

The output printed to the console is shown below, where username is the name of your computer's current user.

Users/username

If the output printed to the console does not match the output format, type the following and then hit Return↩︎:

cd ~; pwd

The figure below is an asbtract representation of our current location. Congrats, you are now in your computer's home directory!

Users/
└── username/

Checkpoint: The current location should be inside the username folder directory.


Changing the Current Location

Now that we know where we are, we need to move to where we need to go.

Note: For the purpose of this tutorial, we will be working in your Desktop directory. If you would like to learn more about how to move through other directories, please see [insert link here].


To change our current directory to the Desktop, we can use cd.

Type the following text into the terminal prompt and then hit Return↩︎:

cd Desktop/custom/

If you would like to confirm you are in the Desktop directory, type the following text into the terminal prompt and then hit Return↩︎:

pwd

Output:

/Users/username/Desktop/custom

The figure below is an abstraction of our current location. We are now in the custom folder directory.

Users/
└── username/
    └── Desktop/
        └── custom/
            ├── icon.png

Checkpoint: The current location should be inside the custom folder directory.

Changing a Folder's Default Icon

Generating a Custom Icon for a Folder

Now that we are in the right location, we can now start converting icon.png into a usable icon for the custom folder.

To convert the image file, you must do the following:

  1. Transform the .png file into its own icon by typing the following text into the terminal prompt and then hit Return↩︎:

    sips -i icon.png

  2. Type the following text into the terminal prompt and then hit Return↩︎:

    DeRez -only icns icon.png > tmpicns.rsrc

    This command transforms the .png file into an .icns file and points to the newly converted file using the tmpicns.rsrc file.

  3. Type the following text into the terminal prompt and then hit Return↩︎:

    Rez -a tmpicns.rsrc -o Icon$'\r'

    This command generates the finalized icon file Icon? as a usable folder icon.


Checkpoint: The custom image file Icon? should be generated inside the custom folder.


Attaching a Custom Icon to a Folder

With the correct icon file generated, we can now attach it to the folder itself so it can be displayed.

To attach the icon and set it visible, type the following text into the terminal prompt and then hit Return↩︎:

SetFile -a C .

Checkpoint: The custom image file Icon? should now be visible as the custom folder's new icon.


Hiding a Custom Icon in a Folder

To avoid any accidental deletion of the Icon? file when working within your newly customized folder, we need to hide the file.


To hide the Icon? file, type the following text into the terminal prompt and then hit Return↩︎:

SetFile -a V Icon$'\r'

In addition, we can use rm to remove the icon.png and tmpicns.rsrc files will keep your newly customized folder tidy.


To remove the files, type the following text into the terminal prompt and then hit Return↩︎:

rm tmpicns.rsrc icon.png

Checkpoint: The custom image file Icon? should now be invisible in the custom. The files tmpicns.rsrc and icon.png should also be gone from the folder.


Removing a Custom Icon from a Folder

If at any point you wish to remove the custom icon, you can do so by removing the hidden Icon? file.


After navigating back into your custom folder directory, type the following text into the terminal prompt and then press Return↩︎:

rm Icon$'\r'

The Icon? file will be removed from the custom directory, allowing the folder to return to its default blue folder icon.
